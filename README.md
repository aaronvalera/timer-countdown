## 🚀 About the Project
This project is an interactive countdown timer designed around a rocket launch theme. It features a professional dark-themed user interface with neon accents, utilizing modern CSS techniques like glassmorphism over a dynamic video background. It showcases practical web development skills through DOM manipulation, interval-based timing functions, and fully responsive web design.

## 🕹️ How it Works
The application simulates a countdown sequence for a space launch. 

**How to use the timer:**
1. Choose and click on one of the preset time buttons (5, 10, 30, 60, or 90 seconds).
2. The selection menu will seamlessly disappear, and the countdown will immediately begin displaying in large, neon-styled numbers.
3. The application precisely counts down second by second.
4. Upon reaching zero, the liftoff sequence triggers: the background video begins playing, the interface announces "LAUNCH!", and the timer UI clears out to give you a full view of the takeoff. 🚀

## 🛠️ Technologies Used
This project was built using standard frontend web technologies:
* 🟦 **CSS (~54.8%):** Handles the layout, responsive media queries, and the premium glassmorphism styling with customized neon hover effects.
* 🟨 **JavaScript (~17.0%):** Manages the countdown logic (`setInterval` and `clearInterval`), dynamic user interface updates, and video playback triggers.
* 🟧 **HTML (~28.2%):** Provides the semantic structure and independent layering for the video background and the main interactive content.
